Australian energy accounts dashboard

Power BI

Built an interactive Power BI dashboard on ABS energy data covering 7 industries and 9 energy products from 2010–2024, showing Australia's shift away from fossil fuels over 14 years.

  • Power Query: filtered totals, replaced nulls, unpivoted year columns, split delimiter, renamed products removing ABS footnote markers
  • DAX: Total Energy, % of Total Energy, Fossil Fuel Flag calculated column classifying products into fossil vs other
  • Slicers: Year, Product, Industry with cross-visual filtering
  • Stopped Year column summarisation in model view to prevent incorrect aggregation

Retail store sales dashboard

Power BI

End-to-end project starting from a deliberately messy dataset. Heavy Power Query cleaning before building a sales and profitability dashboard with a proper date table for time intelligence.

  • Capitalised category column, trimmed all columns, corrected data types

AFL games Excel dashboard

Excel

Interactive Excel dashboard analysing 2,879 AFL games from 2012–2025 — attendance trends, team win rates, venue comparisons, and scoring patterns across 7 pivot tables with Year and Venue slicers.

  • Winner column: =IF(HomeScore>AwayScore, HomeTeam, IF(HomeScore<AwayScore, AwayTeam, "Draw")) — dynamic team name lookup
  • Attendance tier: =IF(J2<30000,"Low", IF(J2<50000,"Medium","High")) — feeds attendance category pivot
  • WinnerType column hardcoded "Home Team" / "Away Team" / "Draw" for home advantage analysis
  • Turned off GetPivotData to enable clean KPI card cell references using =TEXT(ref,"#,##")
  • Custom round sort order applied via Data → Sort → Custom List to fix alphabetical pivot ordering
Pivot tables XLOOKUP IF formulas KPI cards Slicers Conditional formatting
